## Quick Reference

| Element | Markdown |
|---------|----------|
| Italic | `*text*` |
| Bold | `**text**` |
| Code | `` `code` `` |
| Link | `[text](url)` |
| Blockquote | `> quote` |
| Code block | ``` triple backticks ``` |
| Heading | `# Heading` |
| Break | `---` |
| Bullet list | `- item` or `* item` |
| Numbered list | `1. item` |
| Nested list | indent with 2 spaces |
